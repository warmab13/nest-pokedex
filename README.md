# Execute in development

1. Clone repository.
2. Execute
```
yarn install
````
3.- Have Nest CLI installed
```
npm i -g @nestjs/cli
````
4.- Get up database
```
docker-compose up -d
```

5.- Rebuild database using seed
```
http://localhost:3000/api/v2/seed
```

## Used Stack
* Mongodb
* Nest
